Preferences Icons

Preferences icons must be supplied in both @2x and @3x variants to ensure device compatability. All files must be suffixed with @2x.png for a 58x58 pixel size image and @3x.png for a 87x87 resolution respectively.

For example, if you wanted to replace the Airplane icon in your theme you would need the following two files in the afformentioned folders:
AirplaneMode@2x.png
AirplaneMode@3x.png

Please Note: Icons masks do NOT apply to preference icons, you will need to add transparency to each icon in your respective image editor if you wish to mask preference icons.
